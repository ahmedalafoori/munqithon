# munqithon
منقذون
"منقذون" هو موقع إلكتروني ذكي يهدف إلى تطوير خدمات الرعاية البيطرية في السعودية من خلال ربط العيادات البيطرية والأطباء البيطريين المستقلين والمنظمات الخيرية البيطرية في منصة واحدة. يقدم الموقع خدمات متعددة مثل الاستشارات البيطرية، حجز المواعيد، التبني، ونصائح وإرشادات للرعاية بالحيوانات. كما يحتوي على مساعد ذكي (Vetbot) يعتمد على الذكاء الاصطناعي لتقديم تشخيصات أولية مجانية، وخدمة استشارات فيديو مدفوعة، ونظام دفع بنكي بين العميل والطبيب. يوفر الموقع واجهات مخصصة لكل من العميل والطبيب والعيادة، مع نظام متكامل لإدارة المواعيد والتواصل عبر الدردشة، مما يسهم في تحسين تجربة المستخدم ورفع جودة الرعاية البيطرية، بالإضافة إلى تمكين المرأة وتوفير فرص عمل في هذا القطاع.
